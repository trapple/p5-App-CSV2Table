# NAME

App::CSV2Table - print ascii table from csv 

# SYNOPSIS

    csv2table your_csv_file.csv

    or

    nkf -w your_csv_file | csv2table


# DESCRIPTION

App::CSV2Table is a CUI software for pirnting ascii table from csv
which accepts a filename or stdin.
